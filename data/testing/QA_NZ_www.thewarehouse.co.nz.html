How to get a verified address?

A verified address is one selected by using the address finder tool which will enable you to select one of the NZ Post validated addresses from a suggested results list.

What is a verified address?

A verified address is an address listed in the NZ Post database supplied to us. These addresses are known to be correctly formatted and correspond to verified deliverable locations. Both verified and unverified addresses can be added and used for delivery.

Unverified address

A verified address is an address listed in NZPost address database. Your address couldn't be verified, which could lead to delivery delays or lost packages. Please confirm the address you entered.
